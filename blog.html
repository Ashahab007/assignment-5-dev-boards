Question-1: What are the different ways to select an element in the DOM?

The ways to select the different elememt in the DOM are following:

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. querySelectorAll
  5. querySelector

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. innerHTML = get the text with HTML tag
  2. innerText = get only the text content with styles
  3. textContent = get only the text content without styles

Question-3: What is event delegation in the DOM?

Event delegation is controlling the DOM element from it's parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is capturing the target element from the DOM tree and bubble up to the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element using document.createElement(). To add an element using document.classList.add(). To remove element using document.classList.remove()
